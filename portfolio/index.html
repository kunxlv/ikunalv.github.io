Project Details...
VSCODE X TRELLO - A VSCode Extension for Trello
Technology used:   VSCode Extension Framework, Typescript, Trello APIs.

Source Code:   Github

What is the project about?
  • Extracts the TODO comments from the current file.
  • Pin the selected comments from the list of comments to the trello board straight from VSCode.

Features
  • After selecting the comments, you can add a label to it.
  • Provides the exact location of the comment in the Trello card description.
Usage:

1. Initially you need to authorize the extentsion with your trello credentials using "Authorize trello" command. Then use the "Set token" command to set the token. Now, you can start pinning your comments using the "Pin TODOs on Trello" command.


2. Select the comments which you want to pin from all the extracted comments from the current file.

3. Select a board.
4. Select a list.
5. Select a label.
6. Your tasks are created!
7. You can view the exact comment location is the description.
